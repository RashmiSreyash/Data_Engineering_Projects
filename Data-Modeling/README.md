# Data Modeling with Cassandra

## Overview
Designed a NoSQL data model using Apache Cassandra to support specific analytical queries on music streaming data.

---

## Problem Statement
- Traditional relational databases are not efficient for high-scale, query-specific workloads.  
- The goal was to design a query-driven NoSQL data model optimized for performance.

---

## Approach
- Identified key analytical queries
- Designed tables based on query patterns
- Used denormalization for performance optimization

---

## Key Features
- Query-based data modeling
- Optimized partition keys and clustering columns
- Processed CSV data into Cassandra tables
- Ensured fast query performance
