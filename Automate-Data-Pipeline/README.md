#  Data Pipelines with Apache Airflow

##  Overview
Built an automated ETL pipeline using Apache Airflow to orchestrate data ingestion, transformation, and validation workflows.

---

##  Problem Statement
The goal was to automate and monitor ETL workflows for a music streaming dataset, ensuring data reliability and consistency.

---

##  Architecture
- Source → S3
- Processing → Python operators
- Orchestration → Airflow DAG
- Storage → PostgreSQL

---

##  Key Features
- Dynamic DAG creation
- Reusable operators
- Data quality checks
- Scheduled workflows

- scripts/
